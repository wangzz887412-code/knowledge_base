## 🚀 快速开始

### 登录系统

1. 打开网站 http://localhost:5174
2. 使用注册账号登录

### 忘记密码

1. 在登录页面点击"忘记密码？"链接
2. 输入注册时使用的用户名
3. 系统将生成临时密码
4. 使用临时密码登录后，请在设置页面及时修改密码

### 快速上传文档

1. 进入「文档中心」页面
2. 点击上传区域或拖拽文件
3. 支持格式: PDF, DOC, DOCX, TXT, EPUB
4. 文件大小限制: 50MB

## 🤖 AI模型配置

### 支持的模型类型

| 模型               | 提供商   | API密钥   | 特殊能力           |
| ------------------ | -------- | --------- | ------------------ |
| **Ollama本地模型** | 本地部署 | ❌ 不需要 | 免费、隐私、需安装 |
| GPT-4o             | OpenAI   | ✅ 需要   | 高质量、快速       |
| GPT-4o Mini        | OpenAI   | ✅ 需要   | 性价比高           |
| 通义千问VL Plus    | 阿里云   | ✅ 需要   | 支持视觉           |

### Ollama本地模型配置（推荐）

#### 步骤1: 安装Ollama

1. 访问 https://ollama.com/download
2. 下载并安装 Ollama
3. 打开终端，拉取模型:
   ```bash
   # 拉取模型（根据您的硬件选择）
   ollama pull qwen3.5:0.8b    # 推荐：适合低配置电脑
   ollama pull qwen3.5:4b      # 高质量，需要大内存
   ollama pull llama3.2        # 英文为主
   ollama pull deepseek-r1     # 推理能力强
   ```

#### 步骤2: 启动Ollama服务

```bash
# 启动服务（通常自动启动）
ollama serve

# 验证服务状态
curl http://localhost:11434/api/tags
```

#### 步骤3: 在网站中配置

1. 进入「设置」页面
2. 在模型选择下拉框中选择 Ollama 模型
3. 点击「测试连接」验证
4. 点击「保存配置」

### 思考能力（Thinking）

某些模型支持显示AI的思考推理过程：

| 支持思考的模型 | 说明               |
| -------------- | ------------------ |
| qwen3.5 系列   | 通义千问国产强模型 |
| gemma4 系列    | Google模型         |
| deepseek-r1    | 深度求索推理模型   |

**启用方法**:

1. 在设置页面选择支持的模型
2. 找到「💭 思考过程」开关
3. 启用后，AI会显示推理过程

## 📄 文档管理

### 上传文档

1. **拖拽上传**: 将文件拖到上传区域
2. **点击上传**: 点击上传区域选择文件
3. **自动处理**: 上传后自动:
   - 📝 提取文本内容
   - 🤖 生成AI摘要
   - 🏷️ 智能生成标签
   - 🧠 创建思维导图

### 查看文档详情

1. 点击文档卡片
2. 查看标签页:
   - 📋 **AI摘要**: AI生成的文档总结
   - 🧠 **思维导图**: 可视化结构
   - ✏️ **笔记**: 您的学习笔记
   - 📜 **版本历史**: 修改记录

### 处理进度

上传文档后会显示处理进度:

- **0-30%**: 提取文本
- **30-60%**: 生成AI摘要
- **60-100%**: 生成思维导图
- **✅ 完成**: 进度条消失

## 💬 AI助手

### 基本使用

1. 进入「AI助手」页面
2. 在左侧选择要参考的文档
3. 在对话框中输入问题
4. AI会基于选中的文档回答

### 选择AI模型

在AI助手页面顶部可以切换AI模型:

1. 点击模型选择器（显示当前模型名称）
2. 选择想要的AI模型
3. 支持的模型会自动列出

**可用模型**:

- Ollama本地模型（免费、无需API密钥）
- GPT-4o / GPT-4o Mini（需要OpenAI API密钥）
- 通义千问VL Plus（需要阿里云API密钥）

### 选择参考文献

- 左侧边栏显示您的文档列表
- 点击文档即可选择/取消选择
- 支持多文档同时参考
- 选中的文档会高亮显示

### 快捷提示

系统提供常用问题模板:

- 「总结文档内容」
- 「提取关键要点」
- 「基于文档回答问题」
- 「分析核心观点」

### AI思考过程

如果启用思考能力:

- 会先显示AI的思考推理过程
- 然后显示最终答案
- 思考过程可以展开/折叠

## ❓ 常见问题

### Q: Ollama连接失败？

**解决方法**:

1. 确认Ollama已安装并运行
2. 检查服务端口: `curl http://localhost:11434/api/tags`
3. 确认已拉取至少一个模型
4. 重启Ollama: `ollama serve`

### Q: 上传文档后没有生成摘要？

**检查项**:

1. 确认已配置AI模型
2. 如果是云端模型，检查API密钥是否有效
3. 如果是Ollama，检查服务是否运行
4. 查看文档详情中的处理状态

### Q: 处理时间很长？

**说明**:

- Ollama本地模型处理较慢（正常现象）
- 文档越大，处理时间越长
- 可以查看进度条了解状态
- 建议首次使用选择较小的文档

### Q: 如何查看系统日志？

```bash
# 查看后端日志
cd backend
python manage.py runserver
```
